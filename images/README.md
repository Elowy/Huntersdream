# Szimbólum-képek (images/)

Ide töltsd fel a szimbólumok **PNG** képeit az alábbi **pontos** nevekkel.
Ha egy fájl itt van, a játék azt használja az adott szimbólumhoz; ha nincs,
a beépített SVG-rajz marad tartalékként. A képek betöltés után automatikusan
megjelennek — csak töltsd fel őket ebbe a mappába.

## Tippek

- A legtöbb kiszolgáló (pl. GitHub Pages) **megkülönbözteti a kis- és nagybetűt**,
  ezért a fenti neveket pontosan így használd (a játék a kisbetűs változatot is
  megpróbálja, de a fentiek a biztosak).
- Ajánlott **négyzetes** kép (pl. 256×256 vagy 512×512 px).
- Átlátszó (PNG alpha) háttér a legszebb, de a saját hátterű képek is jók.
- PNG mellett a `.jpg` és `.webp` is működik, de a PNG az elsődleges.
